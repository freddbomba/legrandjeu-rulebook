# Le Grand Jeu — Rulebook

English edition of the rulebook for **Le Grand Jeu**, a participatory
future-thinking game co-designed by Federico Bonelli and Raffaella Rovida.

> *Elegantly simple in its design, remarkably complex in its mechanics.*

## Contents

- `rulebook/quickstart-en.html` — 4-page A4 print-ready quick-start rules
- `manual/gm-manual-en.html` — 8-page A4 GM manual, with fold-out cheatsheet
- `player-cards/dashboard-en.html` — 6 per-player A5 reference cards, duplex, 3 × A4 (red/blue/green/yellow/purple/teal)
-  `event-cards/decks-en.html` — 4 × A3 print sheets, 16 cards per sheet (LGJ collective + Fortune personal decks)

## Printing

Open the HTML file in a browser, then print with:

- Paper size: **A4**
- Margins: **None** (or "Default" — the page handles its own margins)
- Scale: **100%**
- Background graphics: **enabled**

Each page maps 1:1 to an A4 sheet.
