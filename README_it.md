# Le Grand Jeu — Manuale

Edizione italiana del manuale di **Le Grand Jeu**, un gioco partecipativo
di visione del futuro co-progettato da Federico Bonelli e Raffaella Rovida.

> *Elegante nella semplicità del design, straordinariamente complesso nella meccanica.*

## Contenuti

- `rulebook/lgj_rulebook_it.html` — regole di avvio rapido, 4 pagine A4 pronte per la stampa
- `manual/lgj_gm_manual_it.html` — manuale del Game Master, 8 pagine A4, con promemoria pieghevole
- `rulebook/lgj_player_cards_it.html` — schede giocatore A5 fronte-retro
- `event-cards/lgj_event_cards_it.html` — fogli di carte evento A3 (mazzi LGJ e Fortuna)

## Stampa

Apri il file HTML nel browser, poi stampa con:

- Formato carta: **A4** (o **A3** per le carte evento)
- Margini: **Nessuno** (o "Predefinito" — la pagina gestisce i propri margini)
- Scala: **100%**
- Grafica di sfondo: **abilitata**

Ogni pagina corrisponde 1:1 a un foglio A4 (o A3).
